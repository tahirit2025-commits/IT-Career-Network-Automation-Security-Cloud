دستاوردهای آموزشی

با مطالعه و دنبال‌کردن مطالب این ریپازیتوری، فرد یادگیرنده با مهارت‌ها و مفاهیم زیر آشنا می‌شود:

شبکه

آشنایی با مدل‌های شبکه، آدرس‌دهی IP، Subnetting، Routing، Switching، VLAN، DNS، DHCP و تکنیک‌های عیب‌یابی شبکه.

مدیریت سیستم‌ها

درک مبانی مدیریت ویندوز سرور و مهارت‌های اساسی در لینوکس.

مجازی‌سازی

شناخت انواع هایپروایزرها، ماشین‌های مجازی، لابراتوارهای عملی و موارد استفاده کاربردی.

پایتون برای IT

یادگیری مبانی پایتون و توانایی اتوماسیون‌سازی وظایف شبکه و امنیت.

امنیت سایبری

آشنایی با اصول امنیت، تحلیل لاگ، تشخیص تهدیدات و مهارت‌های پایه تیم آبی (Blue Team).

محاسبات ابری

درک معماری کلود، مفاهیم AWS و Azure و انجام پروژه‌های عملی مبتدی.

امنیت کلود

آشنایی با ریسک‌های امنیتی در محیط‌های ابری و راهکارهای کاهش و مدیریت تهدیدات.

DevOps و CI/CD

درک مقدماتی از اتوماسیون، مدیریت نسخه، CICD Pipelines و فرایندهای توسعه مدرن.

مخابرات و شبکه‌های Service Provider

شناخت مفاهیم ارتباطی، ساختار شبکه‌های مخابراتی و اصول پایه سرویس‌دهندگان.
